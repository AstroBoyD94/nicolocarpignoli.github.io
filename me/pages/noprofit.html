2019 - ongoing * Open Source project * AR.js Studio I am leading the development and design of the first Web AR Authoring Platform in Open Source. It's an easy-to-use platform to build and host live Web AR apps, without need of coding knowledge. Designed for teachers, students, artists and users who want to start with Augmented Reality for free, with no programming knowledge. It uses AR.js as AR core for tracking and rendering. Built with HTML + CSS + Vanilla Javascript for frontend/backend sides. As of June 2020, AR.js Studio is available in Beta version.

2019 - 2020 * Open Source project * AR.js 3 Development of the first, Open Source, Web AR library for Image Target tracking (NFT). It is now possible, on the Web, to recognize every picture and track them, using the Augmented Reality and to show content.

I contributed on the jsartoolkit5 project, for the NFT (Image Tracking) feature. Later, I brought this technology on AR.js, in order to use it also with framework like A-Frame, to built Web AR experience with Image Tracking using only HTML language.
